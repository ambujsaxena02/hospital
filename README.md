# Hospital Management System (MERN)

A web-based Hospital Management System built using the MERN stack to manage patients, doctors, appointments, and hospital operations with secure role-based access.

## Features
- JWT-based authentication and authorization
- Role-based access (Admin, Doctor, Patient)
- Appointment scheduling and management
- Patient and doctor record management
- RESTful APIs with full CRUD operations
- Responsive React-based UI

## Tech Stack
- Frontend: React.js, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB
- Tools: Git, GitHub, Postman

## Setup
1. Clone the repository  
2. Install dependencies for frontend and backend  
3. Add environment variables (MongoDB URI, JWT secret)  
4. Run backend and frontend servers
